# 首页 `landing.html` · 动效交接手册（给 Fable）

> **这份文件是什么**：把首页这条"动态流程"每个节点、每段动作写死，让 Fable 照着**只做视觉装修**，不用再自己琢磨动线、也别碰坏动作骨头。
> **分工**：Opus 搭了能跑的最简版（动作真动、灰扑扑没装修）+ 这份手册 → Fable 负责把它做漂亮。
> **配套活样板**：本手册的唯一真相是 `landing.html` 本身——边读手册、边跑 `landing.html` 看动作，两个合起来才不含糊。
> **预览**：在 `sandbox3/pages/static/` 目录起 `python -m http.server 8790`，浏览器开 `http://127.0.0.1:8790/landing.html?v=N`（改完务必换 `?v=` 绕缓存）。**必须本地服务打开，不能双击**（要 fetch `head.glb`＋`head-f.glb`）。
> **整体调性（作者拍死）**：黑底 + 冷蓝白粒子的"星空感"，**只有黑白两种颜色**（UI 文字/线框一律白/灰，别加彩色）；星空那版偏暗是**对的、别加稠**。

## 0. 一句话：这页是干嘛的

这是 demo 的**入口页 + 操作台开头**。把项目的核心"蒸馏器"做成肉眼可见的动画：一堆散开的粒子（= 简历/JD/测评等原料）→ 用户把材料喂进中间的终端接口 → 粒子聚拢长成一颗**候选人的数字人头** → 同时左右两侧流式输出"这一幕场景"和"维度报告" → 用户确认后按 Enter，大头缩小转身变成右上角小头**进入操作台**，操作台里依次冒出"选项 → 心里话 → 她的选择 → 理由"。**全程是同一个页面在"变形"，不是几个分开的页面**——这点是产品看点。（注：操作台目前是**草图**、只演了**一个节骨眼**，真版要多幕连演、接真引擎——见第 7 节。）

## 1. 整条流程（状态机总览）

代码里有个 `phase` 变量，四个状态顺序推进：

```
idle（散为材料）
   │  用户拖文件 或 打字+回车
   ▼
distilling（蒸馏中）        粒子从星尘聚拢成头 + 扫描光（凝聚速率 0.00625=2026-06-12 作者拍减半）
   │  （分析完成）
   ▼
formed（已炼成）           头正脸定住；左右两块同时流式长出；两边流完→按 Enter 进操作台
   │  按 Enter
   ▼
entering（进操作台=状态③）  候选人头"变换"（移动+缩放+转背面，不散开）到居中略偏下 → 到位后左上团队头从下往上凝聚
                              → 凝完才打印：顶部场景短句 + 选项→心里话→选中→理由 依次冒出
   │  操作台演完后按 Enter（真版=整场推演跑完自动进）
   ▼
report（总结报告页=状态④）  团队头"融化"（粒子从上往下消失，不散回星空）、候选人头同时反向转身"变换"到总结位；五个数据块依次淡入（01过程/02审计/03拉扯度/04判据表/05建议）
```

> ⚠ **关键真相**：现在 distilling 那 4.5 秒是**计时占位**，不是真分析；左右两块的内容是**硬编码的样本数据**（王先生 × JD158）。真接后端（拖真文件 → 真跑分析）是以后的事，代码里标了 `TODO[接真后端]`。Fable **不用碰这块**，装修时把它当真数据对待即可。

## 2. 逐节点 / 逐转场详解（什么动、从哪到哪、多快、什么手感）

### 节点 A · idle（开局：散为材料）
- **画面**：纯黑底 + 满屏稀疏冷蓝白粒子（头是"散开态"，看着像星空）；左上 `SPOILER` 标；正中一个**白色 `>` + 厚白方块光标**在闪。
- **粒子**：缓慢飘动 + 闲置时整体极轻微左右摆（`idleWob`，±0.10 弧度的正弦摆，不累积）。
- **交互**：① 拖任意文件到页面任意处；② 点中间区域可打字、贴文本。
- **触发下一步**：拖文件松手 / 打字后按回车。

### 转场 A→B · 投喂 → 蒸馏
- 中间终端框 `#portal` **淡出+轻微缩小**（0.55s）。
- 粒子开始**聚拢成头**：`progressTarget` 由 0 → 1，每帧 `progress += (target-progress)*0.04` 缓动靠近（约 2–3 秒聚完）。
- 聚到差不多（`progress>0.85`）时，一道**扫描光**从头顶扫到下巴（周期 12s、扫程 8.5s）。**2026-06-10 作者拍：扫描光从此常驻**——界面 2/3/4（蒸馏完/操作台/结算）所有人头（候选人+团队）都带同速循环扫描，同一道光同步扫。
- **转头与星空已解耦（2026-06-12 作者拍"往回扯"二修去根）**：着色器里 yaw/pitch **只旋转头部目标位，散开星野不吃转头**——凝聚开始那一刻 yaw/pitch 直接钉到 `frontYaw=-1.045 / frontPitch=0.055`（散开态钉值零可见），头长出来就是正脸、全程只凝不拧。连带：**界面1 散开态拖拽不再转动星空**（头炼成后拖拽旋转照旧）；星空的闲置漂移走 `uSpin` 自旋通道、不受影响。

### 节点 B · formed（已炼成：左右流式）
- 头**正脸定住**在屏幕正中、**这个大小不动**。
- **同一刻**，左右两块同时开始**终端风格打字机流式**长出（`startReveal()` 并发跑 `streamScene()` + `streamReport()`）：
  - **左块 `#scenePanel`**（场景待确认卡，**2026-06-11 作者拍重构：名称在前、介绍在后**）：逐字敲出顺序＝**场景名称**（大标题 34ms/字）→ **类别行**（`类别 ∷ 压力测试`，等宽小字 12ms/字）→ 停 0.16s → **情境介绍**（17ms/字）→ 停 0.16s → **核心冲突行**（亮一档 15ms/字）→ **团队成员行**（发丝线分隔的等宽小字 12ms/字，列成员+职位、对应操作台里的团队头；2026-06-11 作者批注：不叫"对手盘"）。打字时末尾有闪烁白方块光标（`.tw::after`）。字段对齐引擎转折点 `title/category/sketch`＋名单 cast。
  - **右块 `#reportPanel`**（**测评画像表 9 维**，2026-06-11 厘清身份：＝AI 按 JD 出情境迫选题→候选人 agent 作答→`quiz_answer` 计分的真产出，真样张 `output/cases/JD158__CV1/画像.md`；**倾向**＝全好题选中项的风险端均值、**置信**＝同维多题作答稳定度（答案飘＝低＝可能是装的）——给"答卷里哪些可靠、哪些可能是编的"参考）：3 列＝维度/倾向/置信；表脚**只留一行读法**（倾向＝答出来的样子·置信＝作答稳定度；材料清单行和诚实脚注行 2026-06-11 作者拍删——**界面 2 不再带诚实脚注，全流程的脚注由状态④标题块小灰字承担**；材料口径备忘：**现管线只有简历·JD·答卷，没有背调/面试官评价**，别再写上去）。动效：**先淡入空表框+表头**，停 0.45s，再**一列一列、每列从上到下**逐格填（13ms/字＋格间 26ms）。曾短暂换成"行为手册表"（2026-06-11 已撤：出处列虚构了不存在的材料）。
  - **右下块 `#personaPanel`**（**人设**，2026-06-11 作者拍上屏：蒸馏器主产出，200-300 词第二人称，样例硬编码 `PERSONA`、末句"未知"=evidence or silence 纪律；**分 2-3 段**——密集大段没人看，作者批注定的）：表格逐格填完后停 0.3s → 淡入 → 正文打字（8ms/字）。**终端风（作者批注定）**：开头 `> `（CSS `::before`）、打完末尾**常驻闪烁光标**（`:not(.tw):not(:empty)::after`，同顶部短句那套）。
- **两边都流完** → `revealed=true` → **按 Enter 可进操作台**。~~底部进入按钮~~＝**2026-06-11 作者批注拍删**（按钮+「或按 ↵ ENTER」提示字都没了，唯一入口=Enter 键）。
- **同日批注拍删的全局件**：品牌下的中文副题（`.brand .sub`）、顶部刻度尺（`.ruler.top`，CSS 整套已拔）。

### 转场 B→C · 按 Enter → 进操作台（**本页的高潮**；2026-06-12 作者拍**转场重做**：候选人头只"变换"不散开）
- 触发：`phase==='formed'` 且 `revealed` 时按 **Enter** → `enterConsole()`。
- 流程：按 Enter 那一刻候选人头**立刻起飞**（作者拍：不停顿），信息卡倒放与飞行**同时**进行——①候选人头**变换**（移动+缩放+转身的连续缓动，**不炸开不重凝**）飞到操作台位 → ②到位后**团队头从下往上凝聚**（速度=界面2 凝聚 0.00625）→ ③团队头凝完才开始打印操作台。
- 变换三要素，**都是缓动、不是硬切**：
  1. **头缩小**：`scale` 1 → `0.355`（每帧 `*easeMove=0.06` 缓动）。
  2. **头移位**：屏幕偏移 `(offX,offY)` (0,0) → **(-0.01, -0.14)**（NDC 坐标：x 正=右、y 正=上；终点=大致居中、略偏下——作者 2026-06-10 调参重定，旧文案"右上小头"作废）。
  3. **头转身**：yaw `frontYaw(-1.045)` → `backYaw(2.985)`，俯仰 → `backPitch(0.36)`，转成"从后往前看"的**背面角度**（注：现在是往一个方向转大半圈 ~230° 到位）。
- **右表 `#reportPanel` 淡出消失 + 左卡 `#scenePanel` 也淡出消失**（按 Enter 那刻两块一起淡出，约 0.5s）；**右下人设例外（作者批注定）：倒放流式消失**——文字从末尾往回删（每 8ms 删 3 字，~0.5s 删完），删空才撤场，光标全程闪着。
- **界面 2 钉死位置（2026-06-11 作者调参第三轮）**：画像表 `left:73.45vw; top:8.9vh`、人设 `left:73.45vw; top:69.7vh; width:18.8vw`（两块**左对齐 73.45**）；场景卡保持默认（5vw/29vh）没动。
- 按钮 `#done` 隐去。
- **头变换完成后**（缓动到位：`scale≈0.32` 且 `yaw≈backYaw`），进入【状态③·操作台】——见下节。

### 状态④ · 总结报告页（草图 · 2026-06-10 新增）

## 5. 架构与红线：哪能改、哪**别碰**

- **能随便改（视觉层）**：所有 `<style>` 里的样式、HTML 结构里的文案排版、面板的位置/字体/间距/网格/按钮样子。这是 Fable 的主战场。
- **别碰（动作骨头）**：
  - `<canvas id="c">` 那套 **WebGL 粒子系统**——GLB 解析、撒点、着色器（VS/FS）、`gl_Position` 的**投影数学**（`uScale`/`uOffX`/`uOffY` 缩放移位那段）、渲染循环里的**缓动与状态机**。动一下整颗头就废。
  - **状态机 `phase` 的流转**与各 `streamScene/streamReport/enterConsole` 的**时序逻辑**。你可以改它们**长什么样**，别改它们**何时发生、谁先谁后**（除非作者点头）。
  - **`#err` 报错层不许删/藏**（live 出错必须亮到页面——诚实工程红线）。
  - **WebGL1 着色器小坑**：顶点/片元里同名变量精度要一致；片元统一 `precision highp float`。新加 uniform 注意别引入精度不一致导致**静默链接失败**。
- **数据诚实（红线，接真后端时尤其注意）**：右边那张报告**真实产物自带诚实脚注**（"AI 扮演候选人猜答、非真人作答""不构成对真实结局的预测""本地效度=0"）。现在最简版为了干净**没显脚注**——**正式版必须把脚注补回某处**，不许让人误以为这是"真人作答 / 真实风险预测"。（详见项目红线：落差只描述不打分、不做录用判决。）

## 9. 对不齐时的法子：实时调参（重要）

遇到**用文字说不清、得靠眼睛定**的视觉值（位置、角度、大小、间距、时长、缓动），**别来回猜**——用这套"实时调参"（即 `tweakpane`/`dat.GUI` 那类调参面板的思路），三步：

1. **临时加个能拖的控件**（滑杆，或键盘 ←→/WASD/[ ]）+ **屏幕上实时显当前数值**；
2. **让作者亲手拖到满意**，把数值念给你；
3. 你把那个数 **钉成默认值，删掉临时控件**（临时件标注"调好即删"）。
